
# Cryptography-project



## to run the app
 Create virtual environments and Activate it
   
   
    py -m venv env

    .\env\Scripts\activate
 
install the requirements.txt by : 

      pip install -r requirements.txt
      
Set the starter flask app  :

      set FLASK_APP=application.py 

      
Run the APP by : 
    
    py application.py
